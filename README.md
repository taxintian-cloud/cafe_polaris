# Cafe Polaris - 公式サイト

**Cafe Polaris** は、静かな空間でコーヒーと紅茶を楽しめるカフェをイメージした、シンプルで温かみのある公式サイトです。  
HTML と CSS を中心に構築し、レスポンシブ対応やアクセシビリティにも配慮しています。

公開ページ:  
👉 https://taxintian-cloud.github.io/cafe_polaris/

---

## 📌 特徴

- **シンプルで読みやすいデザイン**
- ログハウス風の温かい色合い
- **レスポンシブ対応**（スマホでも見やすい）
- **アクセシビリティ配慮**（focus outline、label の関連付けなど）
- **Google Fonts（M PLUS Rounded 1c）使用**
- GitHub Pages による公開

---

## 🏗 使用技術

- **HTML5**
- **CSS3**
- Flexbox（一部）
- メディアクエリ（レスポンシブ対応）

---

## 📂 ページ構成

- `index.html`  
  トップページ（Hero、メニュー、店舗情報、お問い合わせ）

- `style.css`  
  全体のデザインを管理

- `cafe.jpg`  
  Hero セクションのメイン画像

- `4wd.jpeg`  
  店長紹介の画像

- `privacy.html`（任意）  
  プライバシーポリシー

---

## ✨ 主なセクション

### 🔹 Hero（トップ）
- カフェの雰囲気を伝えるメイン画像
- キャッチコピーと紹介文

### 🔹 メニュー
- コーヒー、紅茶、ケーキの紹介
- カード型デザインで統一感を演出

### 🔹 店舗情報（Access）
- 営業時間、住所、電話番号などを表形式で表示

### 🔹 お問い合わせフォーム
- 名前、メール、カテゴリ、メッセージ
- 利用規約チェックボックス付き

---

## 📱 レスポンシブ対応

画面幅 768px 以下で以下を調整：

- aside を下部へ移動
- nav の gap を縮小
- hero の余白を調整
- メニューカードを 100% 幅に
